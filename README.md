# DayPlan — Daily Itinerary Planner

A clean, minimal web app for planning your daily schedule as an interactive to-do list.

## Live Demo

[https://ahaanasubberwal-cpu.github.io/to-do-list/](https://ahaanasubberwal-cpu.github.io/to-do-list/)

## Features

- Add tasks with a time, category, location, and notes
- Group tasks into **Morning, Afternoon, Evening, and Night** sections
- **Drag and drop** to reorder tasks within or across sections
- Mark tasks as completed or highlight them as important
- Color-coded cards by category — Work, Personal, Food, Travel, Shopping
- **Timeline view** showing the full day at a glance
- Progress bar tracking how much of the day is complete
- Day summary with total, completed, and upcoming task counts
- **Duplicate yesterday's plan** with one click
- All data saved to local storage — persists across refreshes

## Usage

No installation needed. Just open `index.html` in any browser.

## Tech

Pure HTML, CSS, and JavaScript — no frameworks or dependencies.
